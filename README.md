# LifePulse 💜

Умный помощник для контроля здоровья с интеграцией Telegram Bot.

## 🌐 Сайт

Откройте: https://zhan19na-lab.github.io/lifepulse/

## 📁 Структура проекта

- `index.html` - Главная страница LifePulse
- `lifepulse-landing.html` - Оригинальный лендинг
- `vizitka/` - Визитка-профиль
- `.github/workflows/` - GitHub Actions для развертывания
